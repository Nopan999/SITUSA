Solusi: Install Driver Untuk Menguhubungkan Printer Dengan Komputer

Penjelasan Masalah

Driver merupakan perangkat lunak yang berfungsi sebagai kemudi untuk menjalankan piranti perangkat keras secara terpisah seperti printer, stereo, pen tab dan lain-lain.

Langkah Penyelesaian

  1. Pada tutorial ini akan dilakukan instalasi driver untuk printer sehingga printer dapat terhubung dengan laptop/komputer
  2. Pertama-tama anda harus tahu model dari printer yang akan digunakan, model dapat dilihat Pada bagian depan produk berbentuk stiker, atau bisa menggunakan image search pada google.
  3. Setelah mengetahui model printer, buka google dan ketik download driver epson/canon (nomor seri).
  4. Usahakan untuk download driver pada situs resmi printer.
  5. Buka situs resmi Epson saat akan download Epson akan secara otomatis mendeteksi versi yang kompatibel dengan OS kamputer/laptop kita, kemudian tinggal tekan downlaod.
  6. Kemudian jalankan instalasi seperti biasa dengan menekan next, beberapa driver mungkin meminta anda untuk menyambungkan printer dengan komputer jadi ikuti langkahnya.
  7. Setelah instalasi selesai anda mungkin diminta untuk restart laptop/komputer ikuti langkahnya.
  8. Setelah restart printer langsung dapat digunakan.
