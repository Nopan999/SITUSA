Solusi: Error #Value! saat menghitung Excel

Penjelasan Masalah

Error Value adalah eror yang muncul karena excel gagal dalam eksekusi sebuah perhitungan ada beberapa penyebabnya namun yang paling sederhana adalah format data pada cell bukan number.

Langkah Penyelesaian

  1. Langkah pertama yang perlu diperhatikan adalah memastikan data pada cell memiliki format number bukan text
  2. Pada saat memasukan data secara manual masukan nomor tanpa tanda baca apapun seperti titik, atau koma agar nantinya bisa dihitung oleh excel.
  3. Untuk menambahkan tanda baca klik kanan pada cell, format cell, centang use 1000 separator untuk menambahkan titik pada angka ribuan, anda juga dapat menyesuaikan dengan fromat penomoran yang ingin di gunakan seperti currency/accounting untuk mata uang, percentage untuk persentase dll.
  4. Jika data sudah terisi oleh data excel akan memastikan apakah data sudah berbentuk nomor atau masih text anda bisa langsung mengubah format cell ke number jika terdapat tanda seru, pilih convert text to number.
  5. Jalankan Perhitungan seperti biasa, error dapat muncul jadi pastikan anda menggunkan rumus yang benar
