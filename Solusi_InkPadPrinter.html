Solusi: Printer Indikator Lampu Merah Berkedip dan Tidak Dapat Mencetak

Penjelasan Masalah

Saat tidak bisa mencetak pada printer terutama epson dengan indikator lampu merah Berkedip harus dilakukan reset menggunakan aplikasi pihak ketiga berikut caranya.

Langkah Penyelesaian

  1. Untuk Mengatasi tidak dapat print, printer harus direset menggunakan aplikasi pihak ketiga disini kita akan menggunakan WIC.
  2. Search pada google WicReset download kemudian install seperti biasa.
  3. Jika sudah terinstal buka wicreset kemudian ikuti langkah-langkahnya, untuk selengkapnya dapat melihat pada video dibawah.
